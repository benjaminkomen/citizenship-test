# Citizenship Test

This repository contains a local web application to help users prepare for the Canadian citizenship test through
practice quizzes. It also includes a personal summary of the official Discover Canada study guide.

## Citizenship Test DOJO

A local practice quiz app to help you prepare for the Canadian citizenship test.

### How to Use

1. Open `index.html` in your web browser
2. Click "Start Quiz" to begin a practice session
3. Answer the multiple choice questions
4. See your results and review incorrect answers
5. Track your progress over time

### Features

- Multiple choice questions based on the Discover Canada guide
- Immediate feedback with explanations
- 75% pass threshold (matches actual test)
- Progress tracking saved in your browser
- Export/import your progress as JSON backup
